# Paired RNA-seq Fold Change Calculation

The paired RNA-seq analysis used edgeR's GLM framework to estimate the fold change between WD and NF samples. Rather than simply averaging the fold changes across the four matched pairs, the model first accounts for overall expression differences between patients (some patients may just have higher expression levels overall), and then estimates the WD-vs-NF difference from what remains — the within-pair differences. The key refinement over a simple average is that pairs where expression is measured more reliably (higher counts) contribute more to the final estimate. In practice, for a balanced dataset like this one with four pairs, the result is very close to simply averaging the within-pair fold changes, but the GLM approach handles the statistical properties of count data more rigorously.
